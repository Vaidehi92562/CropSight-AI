# CropSight-AI

CropSight-AI is a machine learning project focused on predicting risky fruit storage conditions and spoilage risk using environmental and storage-related factors.

## Project Files

- `01_data_loading_and_eda.ipynb` – data loading and exploratory data analysis
- `02_cleaning_and_target.ipynb` – data cleaning and target preparation
- `03_model_training.ipynb` – initial model training
- `03_model_training_v2.ipynb` – improved model training version
- `03_real_world_validation.ipynb` – validation on practical/real-world scenarios
- `04_scenario_simulation.ipynb` – simulation of different storage scenarios

## Objective

The goal of this project is to analyze storage-related conditions such as temperature, humidity, light, CO₂, and fruit type to identify whether storage conditions are safe or risky.

## Tech Stack

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Outcome

This project helps in predicting spoilage risk and supports better storage decisions for fruits using machine learning.
